# My_Portfolio
Portfolio Website with Smooth Scrolling & Dynamic Navigation

This repository showcases a responsive and interactive portfolio website built using HTML, CSS, and JavaScript. It features:

Smooth Scrolling Navigation: Effortlessly navigate between sections with smooth scrolling effects.

Dynamic Navigation Highlighting: Active navigation links update as you scroll through different sections.

Responsive Design: Optimized layout for both desktop and mobile devices.

Form Validation: Basic client-side validation for the contact form.

Dynamic Footer: Automatically updates the copyright year.
